# Phase 2 - Cloud Managed Release Scope

## Goal

Launch MCPGate cloud edition with managed onboarding and tenant-safe operations.

## In-scope

- Multi-tenant control plane.
- Managed runtime provisioning per workspace.
- Hosted secrets management integration.
- Usage and health dashboard.

## Out-of-scope

- Advanced enterprise identity federation (phase 2.5).
- Region-specific residency controls (phase 2.5+).
